# 🏭 نظام إدارة المخزن - FIFO

نظام مخزن متكامل يعمل بنظام **FIFO (أول دخول أول خروج)** مبني بـ HTML/CSS/JavaScript خالص بدون أي خادم أو قاعدة بيانات خارجية.

## ✨ المميزات

- **🏷️ تكويد الأصناف** — إضافة وحذف أكواد الأصناف
- **📥 إضافة الأصناف** — تسجيل الواردات مع الأسعار
- **📤 منصرف الأصناف** — تسجيل المنصرفات بحساب FIFO تلقائي
- **📋 كارت الصنف** — عرض كل حركات الصنف مع الأرصدة
- **📅 التقرير الشهري** — تقرير شهري لكل الأصناف
- **📊 التقرير العام** — رصيد أول وآخر المدة للفترة المختارة
- **📥 تصدير Excel** — تصدير كل صفحة لملف Excel
- **💾 حفظ تلقائي** — البيانات محفوظة في المتصفح (localStorage)

## 🚀 طريقة الاستخدام

### تشغيل محلياً
```bash
# ما تحتاج لأي تثبيت — فقط افتح الملف
open index.html
```

### نشر على GitHub Pages
1. أنشئ repository جديد على GitHub
2. ارفع ملف `index.html`
3. اذهب إلى **Settings → Pages**
4. اختر **Branch: main** وحفظ
5. الرابط سيكون: `https://username.github.io/repo-name`

## 📖 طريقة العمل

### نظام FIFO
- عند صرف صنف، يُحسب السعر من أقدم دفعة واردة أولاً
- يُعرض التكلفة الفعلية لكل منصرف بناءً على طريقة FIFO

### حفظ البيانات
البيانات محفوظة في **localStorage** بالمتصفح. لنقل البيانات بين أجهزة، صدّر Excel أو انسخ البيانات يدوياً.
