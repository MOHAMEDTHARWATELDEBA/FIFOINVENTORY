## ✨ المميزات الجديدة
- 🔐 **نظام تسجيل دخول** (يوزر + باسورد) مع تشفير SHA-256
- 👥 **4 مستويات صلاحيات**: Admin / مدير مخزن / مشاهدة / محاسب
- 📆 **قاعدة بيانات لكل سنة** منفصلة على Google Sheets
- 🔄 **ترحيل تلقائي** لرصيد ديسمبر → يناير الجديد
- 📊 **Google Sheets** كقاعدة بيانات سحابية
- 📥 **تصدير Excel** لكل التقارير

---

## 🚀 خطوات الإعداد (مرة واحدة فقط)

### الخطوة 1 — إنشاء Google Sheet

1. اذهب إلى [sheets.google.com](https://sheets.google.com)
2. أنشئ **Spreadsheet** جديد
3. اسمه مثلاً: `FIFO Inventory Database`

---

### الخطوة 2 — إضافة Google Apps Script

1. في Google Sheet، اضغط **Extensions → Apps Script**
2. احذف الكود الافتراضي
3. **انسخ** كامل محتوى ملف `Code.gs`
4. اضغط **💾 Save** ثم اضغط **▶️ Run** على دالة `getUsersSheet` لتهيئة الشيتات
5. وافق على الصلاحيات المطلوبة

---

### الخطوة 3 — نشر كـ Web App

1. في Apps Script، اضغط **Deploy → New Deployment**
2. اضغط ⚙️ بجانب "Select type" واختر **Web App**
3. اضبط:
   - **Description**: FIFO Inventory API
   - **Execute as**: Me
   - **Who has access**: Anyone
4. اضغط **Deploy**
5. **انسخ** رابط الـ Web App (يبدأ بـ `https://script.google.com/macros/s/...`)

---

### الخطوة 4 — ربط الرابط بالبرنامج

1. افتح ملف `index.html`
2. ابحث عن السطر:
```javascript
const API_URL = 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE';
```
3. استبدل `YOUR_GOOGLE_APPS_SCRIPT_URL_HERE` برابط الـ Web App الذي نسخته

---

### الخطوة 5 — رفع على GitHub Pages

1. ارفع ملفي `index.html` و `README.md` على الـ Repository
2. فعّل GitHub Pages من Settings → Pages
3. الرابط سيكون متاحاً من أي جهاز!
